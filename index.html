Welcome to Git Bootstrap

Two useful templates to start your Git project that will help you to be a Git Jedi.

Instructions

1. Put .gitconfig to your user home directory and change placeholders. If ~/.gitconfig is already exist, you can careful append to the end of file.
2. Put .gitignore to the root directory of your Git project. If .gitignore is already exist, you can careful append to the end of file.

Tested on Git 1.9.3 (OSX)
